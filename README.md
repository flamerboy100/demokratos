# DEMOKRATOS 🗳️

**DEMOKRATOS** es un sistema de votación digital insobornable y universal diseñado para ejecutarse sobre el Kernel de Linux, garantizando elecciones transparentes mediante observación internacional y auditoría comunitaria en tiempo real (estilo Bitcoin).

## 🚀 Características Clave
* **Blindaje a nivel de Kernel:** Aislamiento estricto de sensores biométricos para evitar la manipulación del software electoral.
* **Privacidad Absoluta:** Integración de Pruebas de Conocimiento Cero (ZK-SNARKs) para validar al votante sin revelar su identidad.
* **Red P2P Comunitaria:** Cada ciudadano, escuela u organismo internacional puede correr un nodo para vigilar el conteo en tiempo real.
* **Escrutinio Homomórfico:** Conteo matemático inmediato al cierre de las urnas sin necesidad de abrir los votos individuales.

## 📂 Estructura del Proyecto
* `/src/kernel-module`: Código de bajo nivel para la seguridad del hardware.
* `/src/demokratos-node`: Protocolo de red distribuida y consenso.
* `/src/identity-provider`: Módulos de carga de padrones (Árboles de Merkle) y planillas JSON.
* `/docs/dips`: Propuestas de mejora impulsadas por la sociedad.
